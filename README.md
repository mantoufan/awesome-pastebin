# Awesome Pastebin

A curated list of pastebin, snippet-sharing, online clipboard, and lightweight code-sharing services.

This project welcomes pull requests from pastebin operators, users, researchers, and maintainers. If you run a pastebin service or know one that should be listed, please open a PR with the service URL, a short description, and any important notes such as login requirements, encryption, APIs, CLI support, or shutdown status. We will review and include suitable submissions as quickly as possible.

一个 pastebin、代码片段分享、在线剪贴板和轻量代码分享服务的精选列表。

欢迎各类 pastebin 服务方、用户、研究者和维护者提交 PR。如果你运营或发现了值得收录的 pastebin 网站，请在 PR 中提供服务地址、简短介绍，以及登录要求、加密方式、API、CLI 支持或停运状态等重要信息。我们会尽快审核并收录合适的提交。
